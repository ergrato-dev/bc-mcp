# Ebooks Gratuitos — Semana 09: MCP Client en TypeScript

## TypeScript

- [TypeScript Deep Dive — Basarat](https://basarat.gitbook.io/typescript/) — Libro gratuito online muy completo sobre TypeScript moderno. Capítulos relevantes: Types, Generics, Narrowing
- [Programming TypeScript — Boris Cherny (O'Reilly Preview)](https://www.oreilly.com/library/view/programming-typescript/9781492037644/) — Vista previa gratuita de capítulos introductorios

## Node.js

- [The Node.js Handbook — Flavio Copes](https://flaviocopes.com/tags/node/) — Artículos gratuitos detallados sobre módulos de Node.js (child_process, readline, streams)
- [Node.js Best Practices — GitHub](https://github.com/goldbergyoni/nodebestpractices) — Repositorio con +90 buenas prácticas, completamente gratuito
