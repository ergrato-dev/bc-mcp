# Ebooks Gratuitos — Semana 05

## TypeScript

- [TypeScript Deep Dive — Basarat Ali Syed](https://basarat.gitbook.io/typescript/)
  Referencia completa y gratuita online. Cubre desde conceptos básicos hasta avanzados.

- [TypeScript Handbook — Microsoft](https://www.typescriptlang.org/docs/handbook/intro.html)
  La documentación oficial de TypeScript, accesible como guía progresiva.

## Node.js

- [Node.js Guide](https://nodejs.org/en/learn/getting-started/introduction-to-nodejs)
  Guía oficial de Node.js — cubre módulos ESM, async/await, streams.

## JavaScript y Ecosistema

- [You Don't Know JS — Kyle Simpson](https://github.com/getify/You-Dont-Know-JS)
  Serie completa y gratuita en GitHub sobre JavaScript profundo.
  Especialmente útil: "Scope & Closures" y "Async & Performance".
