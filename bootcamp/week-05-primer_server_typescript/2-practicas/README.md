# Prácticas — Semana 05: Primer MCP Server en TypeScript

## Ejercicios Guiados

| # | Práctica | Tema principal |
|---|----------|----------------|
| 01 | [practica-01-mcpserver-basico](practica-01-mcpserver-basico/) | McpServer + StdioServerTransport + tool `add` |

## Instrucciones generales

> 📌 Los ejercicios usan el formato **descomentar código** — nunca TODOs.
> El estudiante aprende descomentando las secciones indicadas en el `README.md`
> de cada práctica y verificando que el código compile y funcione.

```bash
# Para cada práctica:
cd practica-0X-nombre/
docker compose up --build
```
