# Glosario — Semana 11: Testing, Seguridad y Docker

Términos clave de esta semana, ordenados alfabéticamente.

---

### Coverage

Porcentaje de código ejecutado durante los tests

### Fixture

Objeto de pytest reutilizable para setup/teardown de tests

### InMemoryTransport

Transport del SDK TypeScript en memoria para tests sin E/S real

### Path traversal

Vulnerabilidad que permite acceder a archivos fuera del directorio permitido

### Rate limiting

Técnica para limitar la cantidad de requests por unidad de tiempo

### SQL injection

Ataque que inyecta SQL malicioso en queries construidas con concatenación

### create_connected_server_and_client_session

Helper del SDK MCP Python para crear pares server/client en memoria para tests

### pytest-asyncio

Plugin de pytest para testear código async con async/await

### vitest

Framework de testing para TypeScript/JavaScript moderno y rápido
