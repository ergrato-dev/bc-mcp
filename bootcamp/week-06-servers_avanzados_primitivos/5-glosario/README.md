# Glosario — Semana 06: Servers Avanzados — Los 3 Primitivos

Términos clave de esta semana, ordenados alfabéticamente.

---

### @mcp.resource('{param}')

Decorador FastMCP para resource templates con URI variable

### @mcp.resource()

Decorador FastMCP para registrar un Resource con URI fijo

### Context

Objeto disponible en tools de FastMCP con acceso a logging y recursos del server

### Prompt argument

Parámetro requerido u opcional de un Prompt (name, description, required)

### Resource template

URI con variables entre llaves que genera resources dinámicos

### list_resources

Handler MCP que devuelve la lista de resources disponibles

### list_tools

Handler MCP que devuelve la lista de tools disponibles

### read_resource

Handler MCP que devuelve el contenido de un resource por su URI

### server.resource()

Método del SDK TypeScript para registrar un Resource
