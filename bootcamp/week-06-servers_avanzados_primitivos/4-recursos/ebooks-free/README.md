## Lectura recomendada esta semana

Para entender los Resources y Prompts de MCP, es útil leer sobre:

1. **URI Templates (RFC 6570)**: la especificación que define la sintaxis `{param}` usada en resource templates
   - [RFC 6570 — URI Template](https://www.rfc-editor.org/rfc/rfc6570)

2. **JSON-RPC 2.0**: el protocolo base que usa MCP para comunicación
   - [JSON-RPC 2.0 Specification](https://www.jsonrpc.org/specification)
