# Libros y E-books Recomendados — Semana 07

## SQLite

- **SQLite Documentation** (oficial, gratuito)
  - URL: https://www.sqlite.org/docs.html
  - Secciones clave: SQL Syntax, Data Types, WAL mode, PRAGMA statements

- **aiosqlite Documentation**
  - URL: https://aiosqlite.omnilib.dev/en/stable/
  - Explica el API async: connect, execute, executescript, row_factory

## HTTP y APIs

- **httpx Documentation**
  - URL: https://www.python-httpx.org/
  - Secciones clave: Async support, Timeouts, Exception hierarchy

- **Open-Meteo API Reference** (gratis, sin API key)
  - URL: https://open-meteo.com/en/docs
  - APIs: forecast, geocoding, historical weather

- **Open Library API Documentation** (gratis, sin API key)
  - URL: https://openlibrary.org/developers/api
  - Endpoint utilizado: Search API (`/search.json`)
